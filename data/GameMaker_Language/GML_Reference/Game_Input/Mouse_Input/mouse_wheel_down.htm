mouse_wheel_down

This function returns true if the mouse wheel is being rotated downwards and false otherwise.

 

Syntax:

mouse_wheel_down();

 

Returns:

Boolean

 

Example:

if mouse_wheel_down()
   {
   y += 10;
   }

This moves the current instance down the screen if the mouse wheel is rotated upwards.
