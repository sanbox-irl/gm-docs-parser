date_get_hour

This function returns the hour of the given datetime value.

 

Syntax:

date_get_hour(date);

Argument Description
date The datetime to check.

 

Returns:

Real

 

Example:

myhour = date_get_hour(date_current_datetime());

This would set "myhour" to the current hour.
